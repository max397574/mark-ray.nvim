Mark Ray
========
A plugin I made for learning.

Features
--------
* Displays Marks
* Automatically updated
* Line numbers
* File name/beginning of line

Installation
------------
### [vim-plug](https://github.com/junegunn/vim-plug)

```vim
" Vim Script
Plug 'max397574/mark-ray'
```

Usage
-----
```vim
"Start Plugin
:MarkRayStart
"Open Window
:MarkRayShow
"Close Window
:MarkRayClose
"Stop Plugin
:MarkRayStop
```
